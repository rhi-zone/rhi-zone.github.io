## Focus Pattern

Intentional pivot from the prior reincarnate + existence focus. Current active core:

- **crescent** — spite project: "LuaJIT deserves a real ecosystem." Typechecker is one piece; the scope is libraries, tooling, package manager, the works. V2 typechecker reached Phase 3 (types, env, unify, infer, check — 721 assertions) this week.
- **rainbow** — spite project: "why are frontend codebases 30kloc?" Optics-based reactivity — lenses/prisms give the algebra that frameworks lack. Emerged directly from normalize's codebase analysis work.
- **rescribe** — active. ooxml is now effectively complete and serves as a foundation brick; rescribe is building on top of it. Rescribe is the conversion layer; paraphase (route planner) depends on it being solid first.
- **tiltshift** — foundational for both rescribe and reincarnate. Answers "here's an opaque binary, what is it?" — the question both projects hit when facing undocumented formats.
- **normalize** — analytical engine behind the spite. New tooling directly enabled the insights that produced rainbow's design. Not just infrastructure — it's how "why is this codebase 30kloc?" gets answered with evidence.
