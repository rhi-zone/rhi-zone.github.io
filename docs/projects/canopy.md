# Canopy

**Universal UI client for arbitrary data.**

Canopy is an infinitely configurable UI for viewing and interacting with dataâ€”static files, network streams, video, audio, binary formats, and more.

## Key Features

- **Format-agnostic** - JSON, JSONL, protobuf, msgpack, SSE, video, audio, binary
- **Source-agnostic** - Filesystem, HTTP, WebSocket, stdin
- **User-configurable** - Define custom views for any data shape
- **Live updates** - Streaming data with real-time rendering

## Use Cases

- Log viewers and debuggers
- API response explorers
- Stream monitors
- Binary format inspectors
- Custom dashboards
